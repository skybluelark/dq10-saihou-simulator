# ソルバー エキスパートポリシー仕様 (v0.4 ドラフト)

SOLVER_HEARING v0.1 の回答(2026-07-13)を、④b ロールアウトポリシー実装用のルールに整理したもの。
各ルール末尾の (A1) 等は出典設問。誤読があれば指摘を受けて改訂する。§9 は対話セッションで詰める論点。

## 1. フェーズモデル (A1)

セッションを **削り → アプローチ → 調整** の3フェーズで扱う(みだれぬい実装後は「序盤」なし)。

| フェーズ | 定義 | 行動原理 |
|---|---|---|
| 削り | みだれぬい中+最強精神統一で「アプローチに入れる状態」を作るまで | ダメージ量/集中力の効率が最大の手を選ぶ |
| アプローチ | 最強保持の終了〜弱いの精神統一(=着地)まで | 最強で当てられない残り(27以下等)を他パワーで詰める。**弱い統一後に削る手(2倍/3倍/誤差0になり得ない多マス縫い)を打たずに済む盤面を作る** |
| 調整 | 弱いの精神統一後 | マスごとに誤差0か誤差1を狙う(誤差2狙いはレア)。理想は「全マス誤差1を目指せばよい状態」で調整に入ること |

- マスの多いレシピ(体上等)では、弱いを一度通過して2周目の弱いで着地することもある。

## 2. 中核原則

1. **着地(最重要 A4①)**: 弱い精神統一のターンを事前に計画し、そこまでに削り残し(例: 1マスだけ50)を作らない。着地失敗が★3率を最も落とす。
2. **精度の集中力換算 (A1)**: 「1/7で誤差0を引ける手」は集中力3〜4相当の価値。削り効率との天秤では多くの場合削り優先だが、誤差0の作り置きには加点する。
3. **残り値の非対称性 (A1/C5/E3)**: 残り+2(不足)は ほぐし→かげん系で失敗率〜1/14まで修正可能(ただし集中28)。**−2(超過)は1/2でしか縮まらず最悪の残り方**。誤差2が生じる手は価値が低い。放置してよい縫いすぎは−2まで。誤差3以上は放置しない。
4. **縫いすぎ下限 (E2)**: 結果が−5以下になり得る手は打たない。例外: みだれぬい(リターンがほぐし1回コスト16を上回るなら許容)、再生布(−17以降は慎重に。−30程度まで許容の局面あり C1/D2)。
5. **しあげる (A3)**: ★3 or 何も無し。★2妥協ロジックは将来拡張(価格差×★2→★3確率)。

## 3. ぬいパワー別 (B)

| パワー | 削り/アプローチ | 調整 |
|---|---|---|
| 最強 | 多マス削り(効率最優先)。みだれは「後ろに普通が控えている」場合は温存(B1) | (原則ここに残さない) |
| 強い | みだれほぼ打つ。たすき基調(C2) | 10〜13の処理に「強い×かげん」等(A1) |
| 普通 | できるだけみだれに充てる(B3)。たすき基調 | 14〜25の処理。最善手が最も難しい |
| 弱い | アプローチ: 6〜9と4の処理(虹布除く)。縫う場所がなければ「ぬう」で消費最小 | 刻み(9=かげん×2、10=かげん→半かげん等)を残り集中と相談 |
| ？/会心×2 | 普通として扱う(B4) | シフト会心は調整で会心狙いに使う(C8) |

## 4. 特技別 (C)

- **みだれぬい**: 初手〜「2倍が当たると糸ほぐし1回で戻らない可能性が出る」まで(C1)。再生布では回復がヘッジになり使用域が広がる。
- **ライン系**: 判断基準は集中効率のみ。**ヨコぬい/滝のぼり(2マス)は明確な理由なしに使わない**。参考序列: 最強→3マス系、強い/普通→たすき、弱い→ぬう(C2)。
  - 「明確な理由」の実測例: 隣接2マスがともに適正レンジのとき(ノクトルグローブ #15: 114/15の横並びにヨコぬい@普通 / 烈風 #22・#24: 縦2マスに滝のぼり@最強・強)。
- **ねらいぬい**: 基本使わない。4マスでは考慮、6マス光布ではそこそこ使う。37.8%(奇跡針)は「弱いぬうで7〜9」より高確率なので、調整までに誤差0が不足している場合の選択肢(C3)。
- **かげん/半かげん**: 高コスト調整手。誤差0が取れる場面専用。例外=集中余剰時の刻みで期待値を中央に寄せる(C4)。
- **糸ほぐし**: |残り|3以上の縫いすぎは修正、−2は放置可(C5)。
- **しつけがけ**(C6): ①4マス戦で最強統一前の弱パワーに乗せて最強相当化 ②アプローチでノーダメージにターンを消化し着地に向かう ③確定調整(7にしつけ→かげんで誤差1以内確定 / 光布の16にしつけ→発光ターンかげんで4/0/−4)。
- **精神統一**: **基本2回/戦(最強で削り・弱いで調整)**(C7)。例外: みだれのみで削る構成(最強を当てる場所が少ない体上等)、アプローチで11〜12が3マス以上→強い統一、着地失敗時の緊急使用。
  - **再ロック(ロック中の再使用)による固定窓の延長は実在技術**(リプレイ実測 2026-07-13: 最強ロック中の再統一で最強を5連打、終盤の弱いロック延長も使用)。C7の「2回」は「統一フェーズが2つ」の意で、延長を含む使用回数は4回/戦になり得る【解釈 — 要確認】。ソルバーの候補除外(⓪-(2))は撤回済み。
- **ぬいパワーシフト**: ほぼ使わない。調整でのシフト会心狙いと、削りすぎ時のミスフォロー(最強をノーダメージで飛ばす)のみ(C8)。
- **無我の境地**: 最強では使わない(削りプランが崩れる)。それ以外は早めに。使用パワーの実感分布: 弱5割/普通4割/強1割(C9)。

## 5. 布特性 (D)

- **光布**: **発光ターンは流さない**(初回5Tのみ削り優先許容)。発光ターンが有効になるよう前のターンから配置を作るのが本筋(D1)。ねらい×発光の相性(C3)。
  - ※先日の 0/0/1/6 事例の待ち手は「そうなる前に調整すべき」局面での事後の次善手、と整理する。
- **再生布**: みだれの使用域拡大。アプローチで+2/+3(なければ+5/+6)を普通でつぶす。**着地後に1マス赤(縫いすぎ)を残しておくのは定石**(統一やかげんで7,8に戻るケースのケア。2マス赤も概ね可)。10〜12を強いでつぶすのも定石(D2)。
  - **実測例(烈風のころも下 #31、2026-07-13)**: 残り−2のマスへ意図的にぬう(→−9)、直後の再生発動ターン(T33)の回復+13で+4に戻し、かげんで誤差0に着地。**「残り≤0のマスへ縫う」ことが再生布では正当手になる**(回復対象は累積/基準値が最大のマス=赤いマスに誘導できるため、回復先の制御も兼ねる)。
- **虹布**: 会心ターン=多マス特技で効果最大化(2マスしか当たらないならねらいと比較)。半減ターン=高コスト特技(アプローチで固定パワーなら ほぐし/ねらい等)(D3)。

## 7. 実装方針(④b/④c)

- ロールアウトポリシー: フェーズ判定 → フェーズ別ルールで候補を絞り、同格は既存の静的評価でタイブレーク。
- Stage A 候補絞り込み(④c): −5以下になり得る手の除外(みだれ・再生布例外)、ヨコ/滝の抑制、かげん/半かげんの使用条件、精神統一の局面条件。
- 校正ターゲット: 主戦場44レシピ(叡聖・ノクトル・月影・アンテイク・妖精郷・烈風・陰陽博士・瑞雲・隠密。収録済み)のベンチサブセット。基準=実測85〜95%(9マスは高め)。
- 検証: リプレイ一致率(F1。乖離局面の抽出→対話)。

## 9. 要確認(対話セッションの論点)

1. **フェーズ移行の機械的判定**: 「アプローチに入れる状態」「削り完了」を数値でどう定義するか(弱着地ターンから逆算した残り作業量?)
2. **着地計画**: サイクル上のどの「弱い」を着地点に選ぶか。体上の「2周目の弱い」の判断基準
3. **みだれのやめ時の閾値**: 「2倍が当たったら糸ほぐし1回で戻らない」=最小残りマスがいくつになったら?(2倍の最大打点×倍率から逆算?)
4. **普通ターンの最善手**(たすき vs ぬう)の決め手
5. **E1の隙間**: 残り9、11〜14、16〜27 の処理
6. **強い×かげんの使用域**(10〜13処理)の詳細
7. **光布の「発光ターンを有効化する配置づくり」**の具体則(残り5以上を何マス・どの値で残す?)
8. **無我の使いどころ**の判断構造(「調整観点で待っている手」とは)
